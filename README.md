# 2048 AI

A python based bot that plays 2048 game and is based on expectimax Algorithm and some heuristics.

## Installation and Setup

```

git clone https://github.com/Uditgulati/2048-AI.git

cd 2048-AI

pip3 install -r requirements.txt

g++ ai.cpp

python3 main.py

```
